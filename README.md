# Fluent


`fluent-ex` is inspired by [Project Fluent](https://projectfluent.org). Grammar based on [Fluent v0.9.0](https://github.com/projectfluent/fluent).

## Installation

```elixir
def deps do
  [
    {:fluent, "~> 0.1.0"}
  ]
end
```
